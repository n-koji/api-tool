# api-tool

APIのトラブルシューティングを効率化するためのCLIツール。

HTTPリクエストの送信、レスポンスの整形表示、およびエラー原因の簡易分析を行う。

---

## 背景

実務でAPI連携の不具合調査を行う中で、以下の課題があった：

- 認証エラーやパラメータ不備の切り分けに時間がかかる
- 環境依存を排除した再現が難しい
- リクエスト/レスポンスの確認が煩雑

これらを解決するため、本ツールを作成。

---

## 機能

- 任意のAPIへのHTTPリクエスト送信（GET / POST / PUT 等）
- ヘッダの指定（認証トークン対応）
- JSONレスポンスの整形表示
- レスポンス時間の計測
- ステータスコードに応じた簡易エラー分析
- JSONファイル入力対応（@data.json）
